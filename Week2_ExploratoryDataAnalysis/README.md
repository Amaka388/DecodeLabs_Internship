# Week 2: Exploratory Data Analysis (EDA)

## Project Description
This project focused on exploring the cleaned dataset from Week 1 to understand its structure, distribution, and key statistical properties using Microsoft Excel and Power Query.

## Objectives
- Calculate descriptive statistics (Mean, Median, Mode)
- Identify quartiles (Q1, Q3) to understand data spread
- Count total orders and summarize key metrics
- Use Power Query to explore and shape the dataset

## Tools Used
- Power Query
- Pivot Tables
- Quartile Calculations
- Desriptive Statistics
- Microsoft Excel

## Summary
Exploratory analysis was carried out to understand the dataset's central tendencies and spread, including Mean, Median, Mode, and quartile ranges (Q1, Q3). Order counts and other summary metrics were calculated to establish a baseline understanding of the data before deeper analysis.
